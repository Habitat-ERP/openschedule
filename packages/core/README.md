# @openschedule/core

Core schemas and utilities for OpenSchedule statutory rulesets.

This package contains shared source metadata, source trace, manifest, validation, and diff schema utilities used by OpenSchedule packages.

```bash
npm install @openschedule/core
```

Most consumers should start with `@openschedule/za-customs`, `@openschedule/cli`, or `@openschedule/mcp`.
