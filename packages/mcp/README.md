# @openschedule/mcp

MCP server for OpenSchedule statutory ruleset tools.

```bash
npm install -g @openschedule/mcp
openschedule-mcp
```

Consumer ZA customs tools include `za_customs_sync`, `za_customs_lookup`, `za_customs_rates`, `za_customs_estimate`, `za_customs_source`, `za_customs_measures`, `za_customs_duties`, and `za_customs_reliefs`.

Source and schema tools include `discover_sources`, `fetch_sources`, `check_source_status`, `list_schemas`, and `get_schema`.

OpenSchedule builds from official SARS sources fetched into the user's local cache. It does not publish or bundle SARS PDFs, SARS datasets, or shared generated customs rulesets.
